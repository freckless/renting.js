Rutas da app

Para que a acción do controlador poda ser executada debe nomearse action_{{acción}}
Ex. a acción index deberá nomearse action_index no controlador.


 routes = {
     'url': {
         'método': 'controller#acción'
     }
 };
var routes = { '/': { get: 'home#index' },
